# スキルパワー調整 v2（2026-07-23 社長決定）

## 新しい式

```
基本効果量 = (1 + コスト) × 2 − 1 + (条件属性の数 − 1) − 効果割引
```

- 旧式（(コスト+1)×2 − 効果割引）から全体を **−1**
- **条件属性が1個増えるごとに +1**（条件が重いカードはリターンを大きく）
- 効果を持つカードは、効果の強さ分の割引をそのまま維持
  （割引の判断は `2026-07-22_skill_power.md` の表を引き継ぐ）

## 適用方法

新値 = 旧値 − 1 + (条件属性数 − 1)、下限0。attack / guard / heal の50枚が変化。

## 例外

- コンボスタブ (1-A120-C): 社長直接指定の 基本4＋コンボ時+4 を維持

## 結果メモ

- 条件属性1個のカードは一律 −1（軽い連打がわずかに弱く）
- 条件属性3個のカード（ティアグレイス等）は +1（重い条件のリターン増）

## 追記: AP破壊の割引レートを変更（2026-07-23 社長指摘）

「AP1枚 = ダメージ2点」の換算が重すぎた（ライトニングセイバーが素点6→2に沈む）。
**AP1枚 = ダメージ1点** に変更。

| カード | 変更 |
|---|---|
| ライトニングセイバー | 2 → 4 |
| テクノウェイブ | 5 → 7 |
| スモールバン | 8 → 9 |
| スタンショック | 3 → 4 |
| エレキネット | 0 → 1 |

エボエ・雷雲召喚は全体攻撃の除算（÷生存数）が支配的なため据え置き。

## 追記2: 盾のガード技（守属性guard）は+1補正（2026-07-23 社長決定）

「盾」= 条件属性に **守** を持つguardだけ、基本効果量に **+1** する。
対象3枚: ラージシールド6・シールドガード2・ガードアクション1。
（パリィ・飛翔などの守属性でないguardは対象外）

```
守属性guardの基本効果量 = (1 + コスト) × 2 − 1 + (条件属性の数 − 1) + 1 − 効果割引
```
