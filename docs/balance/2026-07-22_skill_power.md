# スキルパワー調整 v1（2026-07-22）

社長指示: **スキルの基本値 = (消費AP + 1) × 2** にする。追加効果があるカードは、効果の強さの分だけ基本値から割り引く。

## 割引の基準（効果の値段表）

| 効果 | 割引 |
|---|---|
| 敵全体攻撃（3体） | 残り予算 ÷ 3（切り捨て） |
| 控え2体への攻撃 | 残り予算 ÷ 2 |
| 対象を選択して攻撃 | -3 |
| 敵アクター変更・ロック | -3 |
| 敵AP増減 1につき | -2 |
| ドロー・チャージ 1枚 | -2（コスト0帯は-1） |
| 防御スキル使用不可 | -2 |
| 控えから使用できる | -2 |
| 属性×Nのダメージ追加 | 期待値ぶん（属性1個 ≒ 2） |
| 復活回復 | -7 |
| support スキル | 効果文が能力の全てなので基本値 0 |
