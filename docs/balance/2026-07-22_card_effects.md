# カード効果の実装（2026-07-22）

第1弾144枚のうち、効果文を持つ **99枚の効果をエンジンに実装** した（キャラ14体＋スキル85枚）。

## 仕組み（設計）

- 効果は「カードID → 効果定義」の**静的レジストリ**（`engine/src/effects/vol1.ts`）。
  コード読み込み時に1回だけ作られ、バトル状態には保持しない。**カードが増えてもバトルのメモリは増えない。**
- 効果はバトル状態を直接さわれず、**検証付きの EffectApi**（約40種の操作）経由でのみ動く。
- 効果の実行は**スナップショット保護つき**。効果がエラーを起こしたら状態を巻き戻して
  「効果スキップ」のログを残し、バトルは続行する（1枚のバグで対戦が壊れない）。
- 効果の連鎖（反射→戦闘不能→誘発…）は深さ8で打ち切り。ログは300件まで（長期戦でもメモリ一定）。
- レジストリに無いカードは今まで通り「効果未実装」ログが出るだけで安全に動く。

## 発見メモ（要確認）

- **イノセントレイ (1-A070) の条件は「聖＋闇」**だが、両方を持つキャラクターが第1弾にいない。
  属性追加カードとのコンボ前提なら意図通り（例: ハスミール＋邪神の呪い）。

| 勝率 | デッキ |
|---|---|
| 78.9% | 雷土の重撃（ビコウ＋ドッソ） |
| 71.4% | 突撃槍衾（ストミー） |
| 64.6% | 斬の勇者（クラウディア） |
| 55.1% | 闇単アグロ（トランザード） |
| 49.1% | 獣と風（アイ） |
| 36.0% | 聖光の癒し（ハスミール） |
| 34.3% | 氷結コントロール（セレーナ） |
| 10.6% | 竜の猛攻（ジエンド） |

- 決着: **全滅62.9% / 山札切れ37.1%**（効果実装前は 26% / 74%。狙い通り「倒して勝つ」ゲームになった）

- バランス上の注目: ビコウの控え無敵が強すぎる可能性。ジエンドの自己トラッシュ＋高コスト竜スキルは弱すぎる可能性。
